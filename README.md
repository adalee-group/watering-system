# 自动浇水系统

这是我们2016年下的项目：一个物联网的案例，主要包含浇水终端，API Server 和Android客户端。终端读取服务器上的计划任务实施自动浇水。

### 版本
1.0

### 主要技术

这个系统目前运用了以下开源项目：

* [Spring Cloud Netflix] - 包含这三个组件Eureka, Hystrix, Zuul
* [MariaDB] - 开源关系型数据库
* [android] - 一个基于Linux内核的开放移动操作系统

同时，这个系统本身也是开源的

### 环境

你要确保你安装了java以及gradle

*	Java 8
*	Gradle 2.13
*	Intellij IDEA

### 可能会加入的额外功能

 - Docker
 - Auto Recovery
